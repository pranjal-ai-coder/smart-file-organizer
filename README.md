#  📂Smart File Organizer

A Python-based desktop tool that automatically organizes files into categorized folders with a clean GUI interface.

---

## 🚀 Features

- 📂 Automatic file organization by type
- 🖼️ Supports Images, Documents, Videos, Programs
- ⚡ Fast and efficient file sorting
- 🛡️ Safe file handling with collision protection
- 🎨 Modern dark-themed GUI with neon green styling
- 🧠 Built with logic-focused automation

---

## ▶️ How to Use

1. Run the script
2. Click **START SCAN & ORGANIZE**
3. Select the folder
4. Sit back — files will be auto organized
